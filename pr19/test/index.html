Практическое занятие №19 — MySQL

Подготовка

  1. Запустить XAMPP: Apache + MySQL
  2. phpMyAdmin: http://localhost/phpmyadmin/
  3. Вкладка SQL — выполнить скрипты из папки sql/

SQL-скрипты

  • sql/myFirstDataBase.sql — база myFirstDataBase, таблица myFirstTable
  • sql/library.sql — база library, таблица books

Консоль MySQL (Linux)

/opt/lampp/bin/mysql -u root

Проверка через PHP

  • myFirstDataBase / myFirstTable
  • library / books

Примеры команд из задания

SHOW DATABASES;
USE library;
SHOW TABLES;
DESCRIBE books;
SELECT * FROM myFirstTable WHERE myAge > 40;
UPDATE myFirstTable SET myAge = 35 WHERE myName = 'Ольга';
DELETE FROM myFirstTable WHERE myName = 'Sergey';
